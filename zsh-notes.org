* Interesting snippets / idioms

use of ? expansion (which is the many expansions is this?) to produce newline separated path:

#+begin_src zsh
❯ type path

#+begin_src zsh
❯ typeset -r

'?'=0

#+end_src

to see the attributes of a parameter, use +m

#+BEGIN_SRC ZSH
❯ TYPESET +M PATH
ARRAY PATH
#+END_SRC

show parameters with the uniqueness attribute declared:

#+begin_src zsh
❯ typeset -U +
_comp_assocs
cdpath
fpath

#+end_src

show the types of parameters matching a pattern:

#+begin_src zsh
❯ declare +m "*path"
array cdpath
array fpath
array mailpath
array manpath
array module_path
array path
#+end_src
